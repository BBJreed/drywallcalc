# Drywall Calculator Pro - Asset Setup

## Required Assets

Place the following image files in this `assets/` directory:

### 1. icon.png
- **Size:** 1024x1024 pixels
- **Description:** Main app icon with T-square and calculator on light blue (#8BB6D1) background
- **Format:** PNG with transparency support

### 2. splash.png
- **Size:** 1284x2778 pixels (or similar high-res)
- **Description:** Splash screen with light blue (#8BB6D1) background
- **Format:** PNG

### 3. adaptive-icon-foreground.png
- **Size:** 1024x1024 pixels
- **Description:** White T-square and calculator icon on transparent background
- **Format:** PNG with transparency
- **Android:** Used as foreground layer for adaptive icon

### 4. adaptive-icon-background.png
- **Size:** 1024x1024 pixels
- **Description:** Solid light blue (#8BB6D1) background or the full icon design
- **Format:** PNG
- **Android:** Used as background layer for adaptive icon

### 5. favicon.png
- **Size:** 32x32 or 48x48 pixels
- **Description:** Small icon for web browser tab
- **Format:** PNG

## How to Add Assets

1. Save your image files with the exact names listed above
2. Place them in the `assets/` folder
3. Ensure they meet the size requirements for best quality
4. Run `expo start` to test the app with your assets

## Image Specifications

- Use PNG format for all assets
- Maintain transparency where applicable
- Use the brand color: #8BB6D1 (light blue)
- Icon should feature T-square tool + calculator symbol
- Keep design simple and recognizable at small sizes
